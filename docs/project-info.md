# Projektinfo

## Projektname

**Dokumentenmanager**

## Kurzbeschreibung

Der Dokumentenmanager ist eine Webanwendung zur sicheren, strukturierten und durchsuchbaren Verwaltung digitaler Dokumente. Das Projekt verbindet Dateiablage, relationale Metadaten, OCR-Volltextsuche, Kategorisierung, Versionierung, Favoritenverwaltung und Papierkorblogik zu einem konsistenten Gesamtsystem.

## Ziel der Dokumentation

Die Dokumentation soll als technische Referenz des Projekts funktionieren. Sie richtet sich an Entwickler und technisch versierte Dritte, die den Aufbau, die Inbetriebnahme und die Architektur des Systems nachvollziehen möchten.

## Technologie-Stack (Kurzübersicht)

| Bereich | Technologie |
|---|---|
| Backend | Python, FastAPI, SQLAlchemy, Alembic |
| Datenbank | MariaDB / MySQL |
| Web-UI | Jinja2, HTML/CSS/JS |
| OCR | Tesseract, Poppler, pypdf |
| Sicherheit | Fernet, JWT, Bcrypt |
| Dokumentation | MkDocs (Material Theme), GitHub Pages |
