# Dokumentation – Dokumentenmanager

Diese ZIP enthält die überarbeitete, GitHub-Pages-fähige Projektdokumentation des Projekts **Dokumentenmanager**.

## Lokale Vorschau mit MkDocs

```bash
pip install mkdocs mkdocs-material pymdown-extensions
mkdocs serve
```

Danach im Browser öffnen:

```text
http://127.0.0.1:8010
```

## Build

```bash
mkdocs build
```
